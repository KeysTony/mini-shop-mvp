# MVP Telegram Mini App — MiniShop

Это минимально жизнеспособный продукт Telegram Mini App, поддерживающий:
- Обработку естественного языка (NLP) через ИИ
- Геолокацию
- Поиск товаров рядом
- Покупку через TON
